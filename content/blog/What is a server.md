# What is a Server

So, what's a server. It's just a giant, super powerful computer that you can play GTA VI on. The whole **Server vs Client** nodes is real but at the hardware level they're the same.
A server just has access to more storage, process speed, RAM etc so heavy processing is done on server side (There are a few more uses too but we'll talk about them later)
The server is generally the place where all the important data is stored like user password, website specific data and basically everything you need to actually run your website
